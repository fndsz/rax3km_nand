cmcc_rax3000m_immortalwrt

这是一个利用P3TERX编写的Github action 中国移动cmcc_rax3000m 路由自动云编译固件

基于immortalwrt的openwrt，237大佬的闭源驱动

主分支仅集成网络唤醒、Zerotier、定时重启等

旁路由分支，集成usb支持 samba

请在release自行下载

不提供刷机教程，请利用互联网自行寻找
